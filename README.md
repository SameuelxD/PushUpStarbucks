# PushUpStarbucks
Se desarrollo un proyecto FrontEnd donde se uso basicamente CSS Grid para la en maquetacion principalmente con un poco de Flex-Box para acomodar detalles , el proyecto se divide en partes donde por medio de CSS Grid agregue dos columnas de manera que para enmaquetar sea mas facil.
